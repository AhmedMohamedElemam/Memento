                                           ## Memento ##

 Website for Local and International Football News, matches, etc using (C#.Net MVC, SQL, Html5, Css, JavaScript)

##Prerequisites

 You need to install visual studio software (2012) or higher versions and SQL Server (2008).

##Running the Program

 You must export database files from (Memento\DB) to SQL Server then run the program in visual studio.
